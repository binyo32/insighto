# Insighto

AI-powered people search app. Search any name and let AI aggregate results from multiple search engines into one comprehensive, easy-to-read profile — in seconds.

Available on [App Store](https://apps.apple.com/app/insighto/id6760248540) and [Google Play](https://play.google.com/store/apps/details?id=com.Insighto.Anything).

## Features

- **AI-Powered Analysis** — Advanced AI reads, understands and summarizes data from dozens of sources into one clean profile.
- **Multi-Engine Search** — Simultaneously queries Google, Bing, DuckDuckGo and more for comprehensive results.
- **Instant Results** — Get a full profile in seconds, no manual browsing needed.
- **Privacy-Focused** — Only aggregates publicly available information. Your searches stay private.
- **Hot News Feed** — Trending news section about people in the spotlight.
- **Profile Sections** — Background, education, career, social media, fun facts, personality insights and more.

## How It Works

1. **Search a Name** — Type any person's name with optional details like location or job.
2. **AI Analyzes** — AI searches multiple engines, reads hundreds of results and compiles the data.
3. **Get the Profile** — Receive a clean, organized profile with background, career, social links and more.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
